一、选择器进阶
1.1后代选择器：空格
*作用：根据HTML标签的嵌套关系，选择父元素 后代中 满足条件的元素
*选择器语法：选择器1 选择器2{css}
div p{
            color: red;
        }
*结果:后代全设置样式
1.2 子代选择器：>
*作用：根据HTML标签的嵌套关系，选择父元素 子代中 满足条件的元素
*选择器语法：选择器1>选择器2{css}
 div>a{
            color: red;
        }
*结果:只设置子代样式
1.3并集选择器
*作用:同时选择多组标签,设置相同的样式
*选择器语法：选择器1,选择器2{ css}
        h1,
        p,
        div,
        span{
            color: red;
        }
1.4交集选择器
*作用：选中页面中同时满足多个选择器的标签。
*选择器语法：
        /* 必须是p标签，且添加了box属性 */
        p.box{
            color: red;
        }
    </style>
1.5 hover伪类选择器
*作用：选中鼠标悬停在元素上的状态，设置样式。
*选择器语法：选择器：hover{css}
*任何标签都可以添加 伪类标签
1.5 Emmet语法
作用：通过简写语法，快速生成代码。

二、基础选择器
3.1 id选择器 （配合js用）
*定义：#类名{css属性}
        #blue{
            color: blue;
        }
*用法： 在需要改样式的标签中加 id="类名"
    <div id="blue">
        噫吁嚱，蜀道之难难于上青天。
    </div>
*作用：通过类名，找到页面中所有带有这个类名的标签，设置样式。
*注：
  • 所有标签都要id属性
  • id属性值类似身份证，只能一对一使用。
  • 一个标签只能加一个id
  • 一个id选择器只能选中一个标签
4.1通配符选择器
*定义：*{css属性}
            *{
        color: red;
    }
*作用：找到网页中所有标签，设置样式。
*注：一般用于删除内外边距
    * {
    margin: 0;
    padding: 0;
}

5.1 label标签
用法一：
  1. 使用label标签把内容包裹起来
  2. 在表单标签上添加id属性
  3. 在label标签的for属性中设置对应id属性值
性别：
    <input type="radio" name="性别" id="男">  <label for="男">男</label>
    <input type="radio" name="性别" id="女">  <label for="女">女</label>
用法二：
  1. 直接使用label标签把内容和表单一起包裹在一起
  2. 需要把label标签的for属性删去
    性别：
    <label >    <input type="radio" name="性别" >  男    </label>
    <label >    <input type="radio" name="性别" >  女    </label>

一个标签可加多个属性，属性之间用空格隔开，标签和属性之间也用空格隔开，属性之间没有先后顺序。
img标签需要展示对应得效果，需借助标签的属性进行设置。
图片来源src（source）
*同一文件夹直接写名字加后缀即可
替换文本 alt
当图加载失败时，才显示alt的文本。
提示文本 title  
当鼠标悬停时，才显示文本。
宽度和高度  width和height
如果只设置了一个，另外一个没设置会自动等比例缩放。（此时图片不会变形）
